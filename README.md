# My own site! 🚀

This is my personal portfolio website that I built for the Stardance challenge. I wanted to make a clean space to share my coding journey, show off the projects I've built so far, and let people know how to get in touch with me.

## What's on the site?
* **About Me & My Journey:** How I got started back in 6th standard downloading GTA and Minecraft mods, and how I eventually got into learning HTML and Python.
* **My Interests:** Aside from coding, I wrote a bit about my love for chess, strategy/sandbox games, music, and cooking.
* **Projects Box:** A list of things I've built so far, like my password generator, basic calculator, Hangman, and a cool mini-app that tracks who is online in a Minecraft server.
* **Contact:** Quick links to find me on Discord, Instagram, or email.

## Tech Used
* Built entirely with **HTML** for the structure and **CSS** for the custom layout, boxes, and colors.
